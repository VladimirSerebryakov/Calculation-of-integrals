# Учебный проект, выполненый в рамках учебного курса факультета компьютерных наук НИУ ВШЭ
## Собрана многомодульная программа с использованием NASM и языка C. Цель программы - вычисленние корней уравнения и интегралов.

### Постановка задачи
С заданной точностью ε вычислить площадь плоской фигуры, ограниченной тремя кривыми.

**Уравнения кривых:**.
   1. <var>f</var><sub>1</sub> = 0.35<var>x</var>^2 - 0.95<var>x</var> + 2.7,
   2. <var>f</var><sub>2</sub> = 3<var>x</var> + 1,
   3. <var>f</var><sub>3</sub> = 1 / <var>x</var> + 2.

**Метод приближенного решения уравнений:** метод хорд(секущих).
**Квадратурные формулы для вычисления интеграла:** формула прямоугольников. 

Вычисление функций выполнены с помощью NASM. Основная часть программы выполнена на языке C. Добавлены цели для тестов и функции для тестирования. Добавлен разбор опций командной строки
